中国围棋规则（数子法）说明

计分方法：
  • 黑棋得分 = 黑棋棋子数 + 黑棋领地数
  • 白棋得分 = 白棋棋子数 + 白棋领地数 + 贴目
贴目规则：
  • 19路棋盘：7.5子（标准贴目）
  • 13路棋盘：5.5子（考虑棋盘较小）
  • 9路棋盘：3.5子（小棋盘快棋）
术语解释：
  • 领地：完全被己方棋子包围的空点
  • 贴目：补偿白棋后手劣势的额外分数
  • 提子：被对方吃掉的棋子（已从棋盘移除）
